# 📚 EduLens AI – AI Teacher Companion(usig google ai studio)

EduLens AI is an AI-powered learning companion that transforms the way students study. Simply capture an image of a blackboard, notebook, textbook, or handwritten notes, and the app uses AI to extract text, explain concepts in simple language, and generate personalized learning materials.

Powered by **Google Gemini AI**, EduLens AI provides intelligent explanations, multilingual support, AI-generated flashcards, quizzes, revision notes, summaries, and an interactive doubt-solving assistant. Built with **Kotlin**, **Jetpack Compose**, **Material Design 3**, and **Firebase**, the app delivers a fast, secure, and modern learning experience.

## ✨ Features

* 📸 Scan blackboards, textbooks, notebooks, and handwritten notes
* 🤖 AI-powered explanations using Google Gemini
* 🌍 Multi-language support (English, Telugu, Hindi, and more)
* 📝 Generate flashcards, quizzes, summaries, and revision notes
* 🎙️ Voice-enabled AI tutor and doubt solver
* 📊 Study progress tracking and learning analytics
* ☁️ Secure cloud sync with Firebase
* 🌙 Beautiful Material 3 UI with Light & Dark Mode
* 🔒 Secure authentication and user data protection

## 🛠️ Tech Stack

* **Frontend:** Kotlin, Jetpack Compose, Material Design 3
* **Backend:** Firebase Authentication, Firestore, Cloud Storage
* **AI:** Google Gemini API
* **Architecture:** MVVM, Clean Architecture, Hilt
* **Database:** Room & Firebase Firestore

EduLens AI aims to make learning smarter, more interactive, and accessible by turning everyday study materials into an intelligent AI-powered classroom.
